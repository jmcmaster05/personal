# Github CLI

## Reference

* https://github.com/github/hub

## Install

### Mac

    brew install hub

### Windows

    choco install hub

### From Source (for Ubuntu or CentOS)

    cd ~/code
    git clone https://github.com/github/hub.git
    cd hub
    make install prefix=/usr/local
    
## Example Commands

    hub help

    hub clone jmcmaster05/personal

    hub browse jmcmaster05/personal
